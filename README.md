# Detecção de EPI's

## Como executar este trabalho

Para rodar este trabalho, execute os seguintes comandos:

```bash
pip install streamlit
pip install ultralytics
```
Em seguida, inicie a aplicação com:

```bash
streamlit run interface.py
```

Será apresentado uma interface, basta colocar a imagem relacionada a EPI's de construção, assim, será analisada.

Os EPI's analisados nesse trabalho são capacete e colete.
